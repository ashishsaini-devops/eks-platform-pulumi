# 🚀 EKS Platform Setup using Pulumi

This project demonstrates how to provision a production-ready Amazon EKS cluster using Pulumi with Python.

---

## 📌 Overview

This repository showcases how to design and deploy a scalable Kubernetes platform on AWS using Infrastructure as Code.

It reflects real-world platform engineering practices including networking, IAM security, and cluster provisioning.

---

## 🏗️ Architecture

* Amazon EKS Cluster
* VPC with public and private subnets
* IAM roles for cluster and worker nodes
* Managed Node Groups
* Secure authentication and access control

---

## 🛠️ Tech Stack

* AWS (EKS, VPC, IAM)
* Pulumi (Python)
* Kubernetes

---

## ⚙️ Key Features

* Automated EKS cluster provisioning
* Secure IAM-based authentication
* Scalable infrastructure design
* Infrastructure as Code using Pulumi

---

## 🚀 How to Use

1. Install Pulumi and AWS CLI
2. Configure AWS credentials
3. Run:

```bash
pulumi up
```
